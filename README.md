# Frontend Meli

Este proyecto fue generado usando [Nx]

## Servidor de desarrollo

Para correr el servidor de manera local, ejecuta estos comando en terminales separadas:
```
nx serve backend
nx serve frontend
``` 
Entra en http://localhost:4200/. 

## Build

Corre `ng build my-app` para construir el proyecto. 

## Correr unit tests

Corre `ng test my-app` para ejecutar unit tests con [Jest](https://jestjs.io).

## Correr end-to-end tests

Corre `ng e2e my-app` para ejecutar end-to-end tests con [Cypress](https://www.cypress.io).

## Entiende tu espacio de trabajo

Corre `nx dep-graph` para ver un diagrama de dependencias de tu proyecto.
